# **App Name**: LabFlow Pro

## Core Features:

- Centralized Performance Dashboard: Displays key performance indicators such as 'Annual Progress', 'Monthly Progress', and 'Weekly Progress'. Calculates completion rates based on 'cases achieved / total cases'.
- Case & Project Management: Allows tracking and management of individual cases and projects using predefined statuses: 'Assigned', 'In Experiment', 'In Progress', 'Pending Action', and 'Completed (Pending Closure)'.
- Automated Case Prioritization Tool: Automatically analyzes case details as a tool to identify and elevate 'Current Rush / Urgent Cases'. Any cases containing 'FAA' or 'Urgent' keywords are automatically moved to the highest priority display.
- Engineer Workload Monitor & Alerts: Monitors individual engineer case loads. Automatically triggers a 'red high-risk warning' in the summary if any engineer's caseload exceeds 100 cases. Filters '黃顯能' from default assignment/scheduling unless explicitly commanded.
- Equipment & Schedule Overview: Provides a comprehensive view of laboratory equipment and their current availability or scheduled usage.
- PLM Data Sync: Establishes a basic integration with PLM data sources to synchronize 'FAA' and 'FAI' project data into the platform.

## Style Guidelines:

- Primary color: A modern, deep blue (#2842BD) conveying trust and professionalism, for headers and interactive elements.
- Background color: A very light, almost white, cool-toned gray (#F0F1F4) providing a clean, data-friendly canvas.
- Accent color: A vibrant cyan-blue (#30ABF0) used sparingly for critical alerts, call-to-action buttons, and progress indicators.
- All text will use 'Inter', a neutral and highly readable sans-serif font, optimized for screens, ensuring clarity for data-intensive displays.
- Utilize a consistent set of clean, simple, line-art icons that align with data visualization and laboratory themes. Use distinct, high-contrast icons for alerts and key metrics.
- A structured and organized layout, featuring card-based components for dashboards and clear, responsive tables for data lists, prioritizing efficient information consumption.
- Incorporate subtle and swift animations for state changes, data loading, and transitions between views, providing feedback without distracting from critical information.
